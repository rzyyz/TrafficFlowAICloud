# 内测期间注意事项

!!! note 

    内测版本中的一些限制条件，并不代表我们的产品不能够处理相应的情况，而是为了确保大家有较好的体验。针对这些限制条件，我们后续也会不断迭代更新来保障用户体验。

## 识别时长额度有限
内测期间，使用手机号即可登录，每个账号有**一个小时的识别时长额度**

## 内置视频案例
每个内测账号已经默认内置了几个已经识别好的视频，**用户可以直接创建项目**后进行视频配置/分析来体验软件功能

## 上传视频要求
### 视角
- 如果视频拍摄视角是`地面固定视角`：则保障视频清晰无抖动
- 如果视频拍摄视角是`无人机&高空视角`：则最好俯视拍摄，保证飞机完全悬停，拍摄高度最好不要超过130米

### 分辨率和帧率
- 上传的视频最好是`1080p/30fps`，如果有`4K/30fps`、`4k/60fps`或者更高规格视频参数，请使用`剪映/Premier/FinalCutPro`等视频剪辑软件压缩后再上传

## 操作系统要求
内测版仅支持`windows版本`，后续会推出`Mac版本`

## 问题反馈
内测期间，大家有任何问题先查阅[用户手册](用户手册/视频库.md)，手册中若没有对应的说明可以直接在群里提问

## 迭代更新
**内测期间会依据大家的反馈不断更新安装包**
